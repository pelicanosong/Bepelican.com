# Ilustraciones climáticas (3D Fluency)

PNG pre-renderizados por territorio Briselda. Fuente: [Icons8 3D Fluency](https://icons8.com/icon/set/3d-fluency) (atribución en la ficha).

| Archivo | Preset | Uso |
|---------|--------|-----|
| `guajira.png` | `desierto_guajira` | Costa / Guajira |
| `sierra.png` | `sierra_costa` | Sierra / Ciudad Perdida |
| `andes.png` | `andes_urbana` | Bogotá / Monserrate |
| `barrio.png` | `barrio_urbana` | Barrio urbano |
| `nevado.png` | `nevado_alto` | Cocuy / nieve |
| `amazonia.png` | `amazonia` | Amazonía |
| `wx-*.png` | clima en vivo | Icono según OpenWeather |

Para cambiar una ilustración: sustituye el PNG (mismo nombre) o edita `src/lib/climateIllustrationCatalog.ts`.
